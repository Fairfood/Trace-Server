42 by the ``samples`` argument. 

43 :param samples: for `Column` or `SeriesSchema` hypotheses, this refers 

44 to the group keys in the `groupby` column(s) used to group the 

45 `Series` into a dict of `Series`. The `samples` column(s) are

50 column(s) are passed into the `test` function as positional

191 """Calculate a t-test for the means of two samples. 

192 

193 Perform a two-sided test for the null hypothesis that 2 independent 

194 samples have identical average (expected) values. This test assumes 

195 that the populations have identical variances by default. 

196 

197 :param sample1: The first sample group to test. For `Column` and 

198 `SeriesSchema` hypotheses, refers to the level in the `groupby` 

199 column. For `DataFrameSchema` hypotheses, refers to column in 

200 the `DataFrame`. 

201 :param sample2: The second sample group to test. For `Column` and 

202 `SeriesSchema` hypotheses, refers to the level in the `groupby` 

203 column. For `DataFrameSchema` hypotheses, refers to column in 

204 the `DataFrame`. 

205 :param groupby: If a string or list of strings is provided, then 

206 these columns are used to group the Column Series by `groupby`. 

207 If a callable is passed, the expected signature is 

208 DataFrame -> DataFrameGroupby. The function has access to the 

209 entire dataframe, but the Column.name is selected from this 

210 DataFrameGroupby object so that a SeriesGroupBy object is passed 

211 into `fn`. 

212 

213 Specifying this argument changes the `fn` signature to: 

214 dict[str|tuple[str], Series] -> bool|pd.Series[bool] 

215 

216 Where specific groups can be obtained from the input dict. 

217 :param relationship: Represents what relationship conditions are 

218 imposed on the hypothesis test. Available relationships 

219 are: "greater_than", "less_than", "not_equal", and "equal". 

220 For example, `group1 greater_than group2` specifies an alternative 

221 hypothesis that the mean of group1 is greater than group 2 relative 

222 to a null hypothesis that they are equal. 

223 :param alpha: (Default value = 0.01) The significance level; the 

224 probability of rejecting the null hypothesis when it is true. For 

225 example, a significance level of 0.01 indicates a 1% risk of 

226 concluding that a difference exists when there is no actual 

227 difference. 

228 :param equal_var: (Default value = True) If True (default), perform a 

229 standard independent 2 sample test that assumes equal population 

230 variances. If False, perform Welch's t-test, which does not 

231 assume equal population variance 

232 :param nan_policy: Defines how to handle when input returns nan, one of 

233 {'propagate', 'raise', 'omit'}, (Default value = 'propagate'). 

234 For more details see: 

235 https://docs.scipy.org/doc/scipy/reference/generated/scipy.stats.ttest_ind.html 

236 

237 :example:

255 ... ]), 

256 ... "group": pa.Column(str) 

257 ... })

261 ... "group": ["A", "A", "B", "B", "B"] 

262 ... }) 

263 ... ) 

264 >>> schema.validate(df)[["height_in_feet", "group"]]

267 1 7.0 A 

268 2 5.2 B 

269 3 5.1 B 

270 4 4.0 B 

271 

272 """
